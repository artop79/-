Инструкция по обновлению Favicon для проекта Raplle

Вы предоставили логотип Raplle, который идеально подходит для использования в качестве favicon. Я подготовил инструкцию по обновлению всех favicon и связанных с брендом изображений.

Шаг 1: Создание Favicon

Для создания полного набора favicon я рекомендую использовать онлайн-инструмент Real Favicon Generator. Этот инструмент создаст все необходимые версии для различных устройств и браузеров.

Примечание: Вы можете загрузить предоставленный логотип (изображение R на синем фоне) напрямую в Real Favicon Generator, и он создаст все необходимые файлы.

Шаг 2: Загрузка сгенерированных файлов

После генерации favicon загрузите пакет файлов и разместите их в папке /public вашего проекта, заменив существующие:

favicon.ico
apple-touch-icon.png
android-chrome-192x192.png
android-chrome-512x512.png
favicon-16x16.png
favicon-32x32.png
mstile-150x150.png
safari-pinned-tab.svg

Шаг 3: Обновление manifest.json

Обновите файл manifest.json, чтобы он указывал на новые файлы логотипов:

{
  "short_name": "Raplle",
  "name": "Raplle - AI-система для HR",
  "icons": [
    {
      "src": "favicon.ico",
      "sizes": "64x64 32x32 24x24 16x16",
      "type": "image/x-icon"
    },
    {
      "src": "android-chrome-192x192.png",
      "type": "image/png",
      "sizes": "192x192"
    },
    {
      "src": "android-chrome-512x512.png",
      "type": "image/png",
      "sizes": "512x512"
    }
  ],
  "start_url": ".",
  "display": "standalone",
  "theme_color": "#2672e3",
  "background_color": "#ffffff"
}

Шаг 4: Обновление index.html

Обновите секцию head в index.html, чтобы включить все необходимые ссылки на новые favicon:

<link rel="apple-touch-icon" sizes="180x180" href="%PUBLIC_URL%/apple-touch-icon.png">
<link rel="icon" type="image/png" sizes="32x32" href="%PUBLIC_URL%/favicon-32x32.png">
<link rel="icon" type="image/png" sizes="16x16" href="%PUBLIC_URL%/favicon-16x16.png">
<link rel="manifest" href="%PUBLIC_URL%/manifest.json">
<link rel="mask-icon" href="%PUBLIC_URL%/safari-pinned-tab.svg" color="#2672e3">
<meta name="msapplication-TileColor" content="#2672e3">
<meta name="theme-color" content="#ffffff">

Дополнительно: Обновление логотипов в коде

Если в проекте используются встроенные SVG-логотипы, их также следует обновить для соответствия новому бренду.

Примечание по цветам: Судя по логотипу, основной цвет бренда Raplle - градиент от #2672e3 (синий) до #35bfed (голубой). Вы можете использовать эти цвета для обновления цветовой схемы приложения.
